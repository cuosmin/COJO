# Our Shared Life Dashboard

A beautiful, real-time collaborative dashboard for couples to organize plants, meals, budgets, intimacy, and travel together. Built with React and Firebase. **100% free.**

## ✨ Features

### 🌿 Plant Care
- Track all your plants with custom watering schedules
- Get alerts when plants need water
- Monitor plant health levels
- Both partners see updates instantly

### 🍽️ Food & Recipes
- Plan meals together
- Organize recipes
- Track what needs shopping
- Keep your kitchen organized

### 💰 Budget Tracking
- Log shared expenses
- Categorize spending
- See totals at a glance
- Revolut integration ready (optional)

### 💕 Intimacy Reminders
- Schedule quality time together
- Set intentional reminders
- Track what you've done
- Keep connection on the radar

### ✈️ Travel Meal Sync
- When one partner travels, share hotel meals
- Cook the same meal at home
- Stay connected across distance
- Build shared experiences

## 🚀 Quick Start

### 1. One-time Setup (15 minutes)
Follow the **[SETUP_GUIDE.md](./SETUP_GUIDE.md)** for:
- Creating a free Firebase project
- Deploying to Vercel (automatic)
- Getting your live URL

### 2. Share the URL
- Send the Vercel URL to your partner
- You both sign in with Google
- Start using it immediately

### 3. Invite Your Partner
Just share the URL. When you both sign in:
- Your data syncs instantly
- Changes appear in real-time
- Everything stays in sync

## 📋 Tech Stack

- **React 18** — UI framework
- **Firebase** — Real-time database + authentication
- **Tailwind CSS** — Styling (dark, minimal aesthetic)
- **Vercel** — Free hosting
- **GitHub** — Version control

## 💾 Data & Privacy

- All your data lives in your own Firebase project
- Only you and your partner can access it
- No third parties involved
- You control everything

## 🔧 Local Development

```bash
# Install dependencies
npm install

# Create .env.local with your Firebase credentials
cp .env.example .env.local
# Fill in your Firebase values

# Start dev server
npm start

# Build for production
npm run build
```
